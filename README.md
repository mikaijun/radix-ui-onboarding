# 環境構築

依存関係のインストール

```
npm install
```

開発サーバー立ち上げ

```
turbo run dev
```

コードフォーマットチェック(修正無し)

```
turbo run lint
```

コードフォーマットチェック(修正有り)

```
turbo run lint:fix
```

Storybook起動

```
turbo run storybook
```

テスト

```
turbo run test
```
